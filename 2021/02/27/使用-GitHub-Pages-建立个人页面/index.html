使用 GitHub Pages 建立个人页面

新建名字为 <user name>.github.io 的仓库

你需要新建一个名字为 <user name>.github.io 仓库，“user name”是指你的 GitHub 用户名

并且新建之后还有一些配置，在这里不细讲了

使用 Hexo 建立个人博客页面

安装 node.js 和 hexo-cli

打开 node.js 官网下载安装 node.js

打开 Git Bash 输入以下命令安装 hexo-cli

新建网站

新建D:/Blog文件夹，右键 Git Bash Here

输入 hexo init

输入 hexo server 启动 hexo 本地服务

用浏览器访问：http://localhost:4000/ 查看效果
然后，退出本地服务

下载主题

下载一个 Hexo 主题，并放到 themes 文件夹下

编辑 hexo 的全局配置文件： _config.yml 文件

  • 打开 _config.yml 文件修改以下配置项

language: zh-CN 是中文，不写是英文
url: https://<user name>.github.io/（网站地址）
root: / (根目录，GitHub就写斜杠，有些服务器会多一层路径)
skip_render: README.md 这样可以在 /source 里面放一个 README.md，推送的时候自动传到 Github 上面
theme: landscape （主题）
deploy:

type: git
repo: https://github.com/\<user name>/<user name>.github.io（仓库地址）
branch: master（分支）

  • More info: 配置

输入 hexo generate 重新生成静态博客的所有内容
